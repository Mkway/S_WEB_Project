## GEMINI CLI를 이용한 Vibe 코딩

이 프로젝트는 Google Gemini CLI를 활용하여 개발되었습니다. Gemini CLI는 대화형 인터페이스를 통해 코드 작성, 디버깅, 기능 추가 등 다양한 개발 작업을 수행할 수 있도록 돕습니다. 이를 통해 개발자는 아이디어 구상부터 구현까지의 과정을 보다 직관적이고 효율적으로 진행할 수 있습니다.
